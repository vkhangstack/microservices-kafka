# Microservices with Kafka

## Description

This project demonstrates a microservices architecture using Apache Kafka for event-driven communication between services, built with Node.js.

## Features

- Event-driven microservices architecture
- Apache Kafka message broker integration
- Node.js-based services
- Scalable and distributed system design
- Asynchronous communication patternsduplicae

## Prerequisites

- Node.js 22 or higher
- npm or yarn
- Apache Kafka
- Docker (optional)

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start Kafka broker
4. Build and run the Node.js services
5. Test the communication between services

## Architecture

- Service A: Producer service (Node.js)
- Service B: Consumer service (Node.js)
- Kafka: Message broker for inter-service communication
